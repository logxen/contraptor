Tools

This is the list of tools required to make the Contraptor set.

Mechanical

  • A vise (not pictured)
  • Hacksaw
  • Flat file
  • Center punch
  • Starrett 819 automatic center punch (not pictured)
  • Hammer
  • Power drill
  • Drill bits - 7/64", 1/8", 3/16", 1/4", 3/8" (not pictured)
  • Countersink bit (not pictured)
  • Tap 6-32 (not pictured)
  • Screwdriver (phillips, flat)
  • Small screwdriver (flat)
  • Needle nose pliers
  • Combination wrench (3/8", 7/16") - for assembly
  • Hex socket 3/8" (not pictured) - for assembly
  • Allen wrench 1/16" (not pictured) - for assembly
  • Hobby knife
  • Measuring tape
  • Ruler
  • Pencil

Electrical

  • Soldering iron with solder
  • Wire cutters/clippers/crimpers

Other

  • Computer with USB port, printer to print the templates
  • Pegboard at least 2' long
  • Optionally, full sheet labels (e.g. Avery 5265) and adhesive remover (Scotch adhesive remover pen works well)
