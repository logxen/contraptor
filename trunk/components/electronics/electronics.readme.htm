Stepper Motors

The set uses cheap Applied Motion Products unipolar stepper motors 44A501711. They are rated for 2.3A / 2.3V / 60 oz*in. This is technically beyond the limit of the stepper driver board, but because the board has current limiting capability, it works. I do make sure that the driver boards are heatsinked to the frame whenever I run these steppers.

I'm currently searching for a more powerful replacement, because 44A501711 motors seem to stall in certain circumstances when running simultaneously. In addition, I've not been able to find the torque curve for them, so their true parameters remain obscure.

One replacement candidate is RepRap's Keling KL23H51-24-08B motor with twice as much torque. It cost about $25 and can be sourced from MakerBot.com in the US. In Europe, another Reprap's choice - FL57STH51-2808A motor carried by Bitsfrombytes.com - can be an option. 

Stepper Driver Board

The set originally used Reprap stepper driver board v1.1, but the PCB for it is no longer available. You can buy very similar v1.2 Stepper Driver Board from Makerbot.com, which sells kits derived from Reprap designs. One board kit costs $26: http://store.makerbot.com/electronics/electronics-kits/stepper-motor-dri...

The RepRap wiki has detailed instructions for soldering this board. The difference between v1.1 and v1.2 is primarily in layout of components and usage of connectors: v1.2 uses Ethernet sockets for MIN/MAX and IDC with 10-pin ribbon cable for Arduino connection (compare to v1.1). I actually prefer the MIN/MAX connectors in v1.1, and they can still be used with v1.2 board. So, the recommended configuration is SMD v1.2 with MIN/MAX connectors from v1.1.

Alternatively, you can order parts through the parts lister on the Reprap page and etch your own PCBs using these the trace files here: http://forums.reprap.org/read.php?13,8181

You need as many boards as the axes you want to control at a time.

Arduino Controller

You can buy it for little over $30. You can also make or get Freeduino which may be cheaper.

ATX power supply

The PSU can be scavenged from just about any old PC or bought at computer supply stores.
