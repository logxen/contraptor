Limit switch

Overview

Limit switch is a screw mounted on a UHMWPE angle so that it's isolated from the frame. The limit switch wire is connected to the screw. On the board side, a 22kOhm pullup resistor is soldered between +5V (VCC) pin and the limit switch pin. This keeps the limit switch pin at 1 when the switch is open. When sliding element comes into contact with the screw, it grounds the pin to 0. 

Check the wiring scheme for more details.

Materials

UHMWPE channel, 1¼"x¾", 1/8" thick
Machine screw, 10-24
Washers 10
Nut, 10-24
22 kOhm resistor
Сonnector (supplied with stepper board)
Hookup wire

Making

The MIN/MAX switch connector on RepRap stepper driver board (v1.1) has 3 pins - GND, limit switch, VCC (+5V). Solder limit switch wire to the middle pin. Solder the resistor between VCC and middle pins on the connector. Check the schematics and/or board layout to be sure which pin is GND and which is VCC. It's a good idea to use different wire color for MIN and MAX switch (e.g. green, red).

Use the UHMWPE angle pieces leftover from making sliding elements, or make new ones (see Sliding elements section, steps 1-2). Drill 1/4" holes for the limit switch screw and the mounting screw.
