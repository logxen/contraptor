Belt clamp

Overview

The belt clamp consists of the mount, 3/4" screw with nuts, lug with the tightening screw and UHMWPE belt pad.

Materials

Aluminium angle, 1"x1", 1/8" thick
UHMWPE channel, 1¼"x¾", 1/8" thick
Machine screw, 10-24, 3/4"
Machine screw, 6-32, 3/8"
Nuts, 10-24

Source Files

  • This release: components/belt-clamp/

Making

Make the mount and the lug from SVG or PDF template, according to the instructions in Howto section.

Install 3/4" screw into the lug, tighten it with a nut and install 2 more nuts on the screw. Thread 6-32 screw into the 1/8" hole in the lug. Using vise and hacksaw, make two grooves for XL belt in 1/2"x3/8" piece of UHMWPE left over from making sliding elements.
