Belt clamp on sliding element

The belt clamp secures the belt on the sliding element. To install the belt clamp on the sliding element:

Remove the lug with the screw from the belt clamp.

Use needle nose pliers to insert 1/2" screw from inside the sliding element. Install the belt clamp mount on the sliding element, lug hole at the bottom, and tighten the nut.

You can attach the lug, but don't clamp the belt until it's installed on pulleys
